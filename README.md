# Welcome!
I'm so glad you found me! I'm Josh and I still love data — only now I get paid to help other people love it too.

These days I'm an AI/ML educator and builder. I teach Generative AI at Lower Columbia College, run an AI consultancy ([CaellwynAI](https://caellwynai.com)) helping about a dozen clients put LLMs and classical ML to work, and have spent the last four-plus years teaching data science and machine learning to adult learners at Coding Dojo, LearningFuze, and Flatiron School. Before all that, I was a preschool teacher for 7 years — known as "The Toddler Whisperer" — and helped start three project-based schools. Turns out the core idea transfers: people (toddlers, bootcamp students, senior engineers) learn best when they feel seen, heard, and respected, and when what they're learning actually matters.

### 🛠️ What I've been building lately

**🗣️ [Polyglot Forms](https://github.com/Caellwyn) — multilingual voice-AI form system**
Real-time voice form-filling using the Gemini Live API with <50ms barge-in latency and support for 18+ spoken languages. HIPAA-aware architecture keeps PII client-side; piloted with a nonprofit serving recent immigrants. FastAPI + WebSockets + React/TypeScript on Google Cloud.

**🧠 [Long-Memory Character Chatbot](https://github.com/Caellwyn) — RAG-based conversational AI**
A novel architecture that keeps effectively unbounded conversational memory while holding active context under 4,000 tokens. Multi-agent pipeline (main conversation, memory summarization, content-safety filtering) with adjustable persona controls. Publicly deployed on Streamlit.

Plus a production SaaS (PlanPull) that extracts material lists from contractor PDFs at 95%+ accuracy, an ML pipeline that predicts at-risk online students weeks before course end, and a pandemic-era SARIMAX forecasting model that hit <1% MAPE on 30-day infection-rate predictions. Browse the repos below for more.
